Начало работы

Открытие приложения

Заголовок раздела «Открытие приложения»

iAmRich работает как веб-приложение - установка не требуется. Просто откройте в браузере:

iamrich.it-joy.ru

Приложение адаптировано для смартфонов, планшетов и компьютеров.

Авторизация через Telegram

Заголовок раздела «Авторизация через Telegram»

Для входа в приложение используется авторизация через Telegram - это быстро и безопасно.

  1. На главной странице нажмите кнопку Войти через Telegram
  2. Откроется окно авторизации Telegram
  3. Подтвердите вход
  4. Готово! Вы в приложении

Безопасность

Мы не храним ваш пароль - авторизация происходит через защищённый протокол Telegram (HMAC-SHA256). Ваш аккаунт Telegram используется только для идентификации.

Первый расход

Заголовок раздела «Первый расход»

После входа вы попадёте на главную страницу. Чтобы добавить первый расход:

  1. Перейдите в раздел Расходы через боковое меню
  2. Нажмите кнопку + (Добавить)
  3. Заполните форму:
    • Дата и время - когда произошла трата
    • Валюта - рубли, доллары, евро и другие
    • Сумма - стоимость покупки
    • Категория - выберите подходящую (Еда, Транспорт и т.д.)
    • Название - краткое описание (например, «Обед в кафе»)
  4. Нажмите Сохранить

Совет

Попробуйте голосовой ввод! Нажмите на иконку микрофона и скажите, например: «Купил хлеб за 80 рублей». Приложение автоматически заполнит все поля.

Категории

Заголовок раздела «Категории»

Расходы и доходы разделены по категориям:

  • Расходы (красные/оранжевые): Еда, Транспорт, Развлечения, ЖКХ и другие
  • Доходы (зелёные): Зарплата, Фриланс, Подарки и другие

Вы можете создавать свои категории в разделе расходов - нажмите на иконку настройки категорий.

Навигация по приложению

Заголовок раздела «Навигация по приложению»

Основное меню находится слева (на компьютере) или открывается по нажатию на гамбургер-меню (на смартфоне):

  • Главная - общая информация и баланс
  • Расходы - список всех записей с фильтрами
  • Периодические - автоматические повторяющиеся платежи
  • Профиль - ваши данные
  • Настройки - язык и тема оформления
  • Подписка - управление тарифом
  • Справка - эта документация
  • О приложении - версия, контакты, обратная связь
