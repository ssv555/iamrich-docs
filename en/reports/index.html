Reports

Overview

Section titled “Overview”

The Reports section provides summary information about your income and expenses for a selected period. Unlike the expense list, data here is aggregated — you see totals by days, weeks, months, years, or categories.

How to Open

Section titled “How to Open”

In the sidebar, click Reports — a submenu will expand with available reports.

Report by Periods

Section titled “Report by Periods”

Displays a table with income, expense, and balance totals grouped by time intervals. Data is shown broken down by currency.

Important

Only actual transactions are included in the report (not planned or transfers).

Filters

Section titled “Filters”

The top panel includes:

FilterDescription
PeriodDate range selection (from — to). Default — current month
Group byBy days, weeks, months, or years. Default — by days
Page sizeNumber of rows per page

Table

Section titled “Table”

Each table row contains:

ColumnDescription
PeriodDate or interval (e.g., 15.02.2026 or 02.2026)
CurrencyCurrency code, color-coded
IncomeTotal income for the period (green)
ExpenseTotal expenses for the period (orange)
BalanceDifference between income and expenses. Green if positive, orange if negative

Tip

Click a column header to sort. By default, data is sorted by date (newest first).

Totals

Section titled “Totals”

At the bottom of the page is a collapsible Totals panel. It shows overall sums for the entire selected period by currency:

  • Total Income
  • Total Expense
  • Total Balance

Pagination

Section titled “Pagination”

Page navigation is located below the table — similar to the Finances section.

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing period, currency, income, expense, and balance information.

Report by Categories

Section titled “Report by Categories”

Shows where your money goes — top categories for a selected period with transaction counts, amounts, and percentages.

Important

Only actual transactions are included in the report (not planned or transfers).

Filters

Section titled “Filters”

The top panel includes:

FilterDescription
PeriodDate range selection (from — to). Default — current month
CurrencyA specific currency or all at once
TypeToggle: Expense / Income / All. Default — Expense
Page sizeNumber of rows per page

Table

Section titled “Table”

Each table row contains:

ColumnDescription
CategoryCategory name. Green — income, orange — expense
TransactionsNumber of transactions in this category
AmountTotal amount for the category (always positive)
% of TotalShare of the total amount + visual progress bar

Tip

Click a column header to sort. By default, data is sorted by amount (highest first).

Totals

Section titled “Totals”

At the bottom of the page, a panel displays the total transaction count and total amount for the selected period.

Pagination

Section titled “Pagination”

Page navigation is located below the table — similar to the Finances section.

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing category name, amount, transaction count, and a progress bar.

Period Comparison

Section titled “Period Comparison”

Compare income and expenses by category between two arbitrary periods. Clearly shows how your spending structure has changed.

Important

Only actual transactions are included in the report (not planned or transfers).

Filters

Section titled “Filters”

The top panel includes:

FilterDescription
Period AFirst (baseline) period for comparison. Default — previous month
Period BSecond (current) period for comparison. Default — current month
CurrencyA single currency. Default — RUB

Summary Cards

Section titled “Summary Cards”

Above the table, three cards display period-over-period totals:

CardDescription
IncomePeriod B income and % change relative to Period A
ExpensePeriod B expense and % change
BalancePeriod B balance and % change

Arrow color indicates direction: green — positive change, red — negative.

Table

Section titled “Table”

Each table row contains:

ColumnDescription
CategoryCategory name. Green — income, orange — expense
Period AAmount for this category in the first period
Period BAmount for this category in the second period
ChangeAbsolute difference (Period B − Period A) with arrow and color
Change %Percentage change. “New” — if the category only appears in Period B

Tip

Click a column header to sort. By default, data is sorted by change % (highest first).

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing category name, amounts for both periods, change, and percentage.

Balance by Currency

Section titled “Balance by Currency”

Shows cumulative balance per currency across all time — how much money you currently have in each currency.

Important

Only actual transactions are included in the report (not planned or transfers).

Key Features

Section titled “Key Features”
  • No filters — the report automatically includes all transactions from all time
  • No pagination — the number of rows is limited by the number of currencies used

Table

Section titled “Table”

Each table row contains:

ColumnDescription
CurrencyCurrency code, color-coded
Total IncomeTotal income amount (green)
Total ExpenseTotal expense amount (orange)
BalanceDifference between income and expenses. Green if positive, red if negative
Last TransactionDate of the last transaction in this currency

Tip

Click a column header to sort. By default, data is sorted by balance (highest first).

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing currency, balance, income, expense, and last transaction date.

Family Report

Section titled “Family Report”

Shows income and expense breakdown per family member — who earned and spent how much, and what share of total family expenses each member accounts for.

Family Mode Only

This report is only available to users with an active family. If you are not in a family, the “By Members” item does not appear in the menu.

Important

Only actual transactions are included in the report (not planned or transfers).

Filters

Section titled “Filters”

The top panel includes:

FilterDescription
PeriodDate range selection (from — to). Default — current month
CurrencyA single currency. Default — RUB

Summary Cards

Section titled “Summary Cards”

Above the table, three cards display family-wide totals:

CardDescription
Total IncomeCombined family income (green)
Total ExpenseCombined family expenses (orange)
Total BalanceDifference between income and expenses. Green if positive, orange if negative

Table

Section titled “Table”

Each table row contains:

ColumnDescription
MemberAvatar and name of the family member
IncomeMember’s total income (green)
ExpenseMember’s total expenses (orange)
BalanceDifference between member’s income and expenses
TransactionsNumber of transactions
% of TotalMember’s share of total family expenses + visual progress bar

Tip

Click a column header to sort. By default, data is sorted by expense (highest first).

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing avatar, member name, income, expense, balance, and percentage of total.

Regular Expenses

Section titled “Regular Expenses”

Analysis of recurring spending — subscriptions, utilities, and other categories that appear consistently across multiple months. Shows average monthly cost, min/max range, and total for each qualifying category.

Important

Only actual expenses are included in the report (not income, planned, or transfers).

Filters

Section titled “Filters”

The top panel includes:

FilterDescription
PeriodDate range selection (from — to). Default — last 3 months
CurrencyA single currency. Default — RUB
Min monthsMinimum number of months a category must appear in. Default — 2

Summary Card

Section titled “Summary Card”

Above the table, a card displays the estimated monthly fixed expenses — the sum of averages across all regular categories.

Table

Section titled “Table”

Each table row contains:

ColumnDescription
CategoryCategory name
Months ActiveHow many months out of the total the category appeared in (e.g., 3 / 3)
Avg / MonthAverage monthly expense amount (orange)
MinMinimum amount in any single month
MaxMaximum amount in any single month
TotalTotal amount for the entire period (orange, bold)

Tip

Click a column header to sort. By default, data is sorted by average per month (highest first).

Mobile Version

Section titled “Mobile Version”

On mobile devices, compact cards are displayed instead of a table, showing category name, months count, average amount, min, max, and total.

Forecast

Section titled “Forecast”

Projection based on historical averages — shows how much you typically earn and spend per month, and how the current month is tracking against that baseline.

Important

Only actual transactions are included (not planned or transfers). The current incomplete month is excluded from historical averages to avoid skewing results.

Filters

Section titled “Filters”
FilterDescription
Basis PeriodNumber of past months to analyze: 1, 3, 6, or 12. Default — 3 months
CurrencyA single currency to display

Summary Cards

Section titled “Summary Cards”

Three cards at the top show historical monthly averages:

CardDescription
Avg Monthly IncomeAverage income per month over the basis period (green)
Avg Monthly ExpenseAverage expense per month over the basis period (orange)
Projected Monthly BalanceAverage balance per month. Green if positive, orange if negative

Progress Table

Section titled “Progress Table”

Below the cards, a table compares the current month to historical averages:

ColumnDescription
MetricIncome, Expense, Balance, or top expense categories
Current MonthActual value so far this month
Monthly AvgHistorical average for this metric
ProjectedExtrapolated end-of-month value based on days elapsed
DifferenceProjected minus average. Green if favorable, red if unfavorable

Tip

The projection uses a simple formula: current value / days elapsed * days in month. Early in the month the projection may be volatile — it becomes more accurate as the month progresses.

Mobile Version

Section titled “Mobile Version”

On mobile devices, each metric is shown as a compact card with current value, average, projection, and difference.

Spending Anomalies

Section titled “Spending Anomalies”

Detects unusually large transactions and peak spending days based on a configurable threshold compared to your category averages and daily averages.

Important

Only actual expenses are included in the report (not income, planned, or transfers).

Filters

Section titled “Filters”
FilterDescription
PeriodDate range selection (from — to). Default — last 3 months
CurrencyA single currency. Default — RUB
ThresholdMultiplier for detecting anomalies: 1.5×, 2×, or 3× the average. Default — 2×
Page sizeNumber of rows per page (Section A only)

Section A: Unusual Transactions

Section titled “Section A: Unusual Transactions”

A table of individual expenses that exceed the threshold × category average for the selected period.

ColumnDescription
DateTransaction date
CategoryCategory name (orange)
TitleExpense title
AmountTransaction amount (orange, bold)
Category avgAverage expense for this category in the selected period
DeviationHow many times the amount exceeds the average (e.g. 3.2×, red)

Sorted by deviation (highest first). Supports pagination.

Section B: Peak Spending Days

Section titled “Section B: Peak Spending Days”

Top-10 days with the highest total spend relative to the daily average for the selected period.

ColumnDescription
DateCalendar date
Total spentTotal expenses for that day (orange, bold)
TransactionsNumber of transactions on that day
Daily avgAverage daily spend for the selected period
DeviationHow many times the day’s spend exceeds the daily average (red)

Fixed top-10 list sorted by deviation (highest first). No pagination.

Mobile Version

Section titled “Mobile Version”

On mobile devices, each section displays compact cards. Section A cards show date, category, amount, and deviation. Section B cards show date, total spent, transaction count, and deviation.
